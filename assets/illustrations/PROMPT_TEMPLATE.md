# イラスト生成プロンプトテンプレート

## 共通スタイル指示（毎回最初に入れる）

```
以下のスタイルで運動指導用のイラストを1枚作成してください。

【スタイル】
- セミリアリスティックなフラットイラスト
- 白背景、影なし
- 人物: 中年男性、短髪黒髪、白い半袖Tシャツ、紺のハーフパンツ、白い靴下
- 輪郭線: 細い黒線
- 色数: 最小限（肌色、白、紺、青）
- 運動の動作方向を示す赤い矢印（太め、はっきり見える）
- 補助具がある場合は青色で描く（枕、タオルなど）
- 医療的に正確な関節角度と体の向き
- 余計なテキストや文字は入れない
- 横長の構図（アスペクト比 4:3）
```

## 各運動の個別プロンプト

### 1. パテラセッティング
```
【運動】パテラセッティング（Quad Sets）
仰向けに寝た状態。膝の裏に青い丸めたタオルを置いている。
膝裏でタオルを押し潰すように力を入れている瞬間。
太ももの前面（大腿四頭筋）に力が入っている様子が分かるように。
視点: 横から（側面図）、足側をやや手前に。
赤い矢印: 膝が下方向にタオルを押す方向を示す。
```

### 2. SLR（下肢伸展挙上）
```
【運動】SLR（Straight Leg Raise）
仰向けに寝た状態。反対側の膝は曲げて足底を床につけている。
運動側の脚は膝をしっかり伸ばしたまま、約30〜45度持ち上げている瞬間。
視点: 横から（側面図）。
赤い矢印: 足が上方向に持ち上がる方向を示す。
```

### 3. ブリッジ（ヒップリフト）
```
【運動】ブリッジ / ヒップリフト（Hip Lift / Bridge）
仰向けに寝た状態。両膝を曲げて両足底を床につけている。
お尻を持ち上げて、肩から膝まで一直線になっている瞬間。
頭の下に青い枕。
視点: 横から（側面図）。
赤い矢印: お尻が上方向に持ち上がる方向を示す。
```

### 4. クラムシェル
```
【運動】クラムシェル（Clamshell）
横向き（側臥位）に寝た状態。股関節約45度、膝約90度に曲げている。
両足はつけたまま、上側の膝を貝殻が開くように持ち上げている瞬間。
骨盤は安定して後ろに倒れていない。
視点: 正面やや上から（横向きの体を前方から見る）。
赤い矢印: 膝が開く方向（上方向）を示す。
```

### 5. スクワット
```
【運動】スクワット（Squat）
立った状態から、椅子の背もたれに両手を軽く添えて、腰を落としている瞬間（ハーフスクワット）。
足は肩幅に開いている。膝がつま先より前に出すぎていない。
視点: 横から（側面図）。膝・股関節の屈曲角度と体幹の前傾が分かるように。
赤い矢印: 腰が下方向に沈む方向を示す。
椅子はシンプルなパイプ椅子。
```

### 6. カーフレイズ
```
【運動】カーフレイズ（Calf Raise）
立った状態。壁に両手を軽く添えている。
つま先立ちで踵を最大限持ち上げている瞬間。
体はまっすぐ（前傾していない）。
視点: 横から（側面図）。ふくらはぎの筋収縮が分かるように。
赤い矢印: 踵が上方向に持ち上がる方向を示す。
```

### 7. ドローイン
```
【運動】ドローイン（Abdominal Drawing-in）
仰向けに寝た状態。両膝を曲げて足底を床につけている。
おへそを背骨に引き寄せるようにお腹を凹ませている瞬間。
視点: 横から（側面図）。
赤い矢印: お腹が凹む方向（下方向・内側に向かって）を示す。
お腹が凹んでいる様子が明確に分かるように描く。
```

### 8. 座位での膝伸展
```
【運動】座位での膝伸展（Seated Knee Extension）
椅子に座った状態。背筋を伸ばしている。
片脚の膝をまっすぐ伸ばして水平に持ち上げている瞬間。
つま先は天井に向けている（足関節背屈）。
視点: 横から（側面図）。
赤い矢印: 足が前方・上方向に持ち上がる方向を示す。
椅子はシンプルなパイプ椅子。
```

### 9. 肩甲骨内転運動
```
【運動】肩甲骨内転運動（Scapular Retraction）
椅子に座った状態。背筋を伸ばし、顎を引いた良い姿勢。
両肘を軽く曲げて、肩甲骨を背骨に向かって寄せている瞬間。
視点: 後方から（背面図）。肩甲骨が内側に寄っている様子が分かるように。
赤い矢印: 両方の肩甲骨が中央に向かって寄る方向を示す（2本の矢印）。
```

### 10. ウォールプッシュアップ
```
【運動】ウォールプッシュアップ（Wall Push-Up）
壁から腕一本分離れて立っている。両手を肩幅で壁についている。
肘を曲げて体を壁に近づけている瞬間。
体は頭から踵まで一直線を保っている。
視点: 横から（側面図）。肘の屈曲と体の傾斜角度が分かるように。
赤い矢印: 体が壁に向かって近づく方向を示す。
```

## 生成後のチェックリスト

- [ ] 関節角度は医療的に正確か？
- [ ] 左右が正しいか（反転していないか）？
- [ ] 赤い矢印は動作方向を正しく示しているか？
- [ ] 人物の服装・体型は統一されているか？
- [ ] 背景は白か？余計な要素はないか？
- [ ] テキストが含まれていないか？
- [ ] 保存ファイル名: `{exercise-id}.png`（例: `squat.png`）
