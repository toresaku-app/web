# RehabHEP 開発計画

## コンセプト
理学療法士が自主トレ指導書を3分以内で作成・PDF出力できるアプリ

## MVP機能
1. 運動ライブラリ（10種類）の一覧表示・検索
2. 運動の選択・パラメータ設定（回数・セット・注意点）
3. 指導書プレビュー
4. PDF出力・画像保存・共有（LINE等）
5. テンプレート保存・呼び出し

## UXフロー
1. 運動を部位・姿勢でフィルタ → サムネイルから複数選択
2. 各運動の回数・セット・注意点を調整
3. プレビュー確認・並び替え
4. PDF出力 or 画像保存 → 印刷/LINE共有
