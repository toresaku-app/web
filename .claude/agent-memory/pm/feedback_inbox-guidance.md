inbox に書き込んだら、毎回「どのターミナルで inbox と打てばいいか」を順番付きで案内する。

**Why:** 利用者から「毎回教えてくれない？」と要望があった。どの inbox に何を書いたか pm は把握しているが、利用者は複数ターミナルを行き来しているので案内がないと迷う。

**How to apply:** inbox への書き込み完了メッセージに、必ず「次のステップ」として打つターミナルと順番（依存関係があれば順序指定、なければ同時OK）を添える。
