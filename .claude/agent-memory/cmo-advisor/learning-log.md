# cmo-advisor 学習ログ

**相談役モード特有のファイル**。利用者（オーナー）のマーケティング学習進捗を追跡する。

毎セッションの最後、または新しい概念を教えた直後に更新する。

---

## 使い方

このファイルは相談役モードの心臓部。利用者の成長曲線を覚えていることが cmo-advisor の価値を決める。

新しい概念を教えたら「説明済み概念」に追加。理解度を◯/△/×で記録。
宿題を出したら「宿題ログ」に追加。結果が出たら評価を記録。

## 宿題ログ

### 出した宿題と結果

まだ相談役モードに移行したばかりなので、宿題ログは空。今後ここに積む。

**テンプレート**:
```
### 宿題 N: [タイトル]（出した日）
- **依頼内容**: 具体的に何をやってほしいか
- **成功基準**: どうなったら完了か
- **期限**: いつまで
- **結果**: （完了後に記録）
- **振り返り**: 何が効いた／効かなかった
```
