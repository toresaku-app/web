# RehabHEP 起業プラン

> 個人開発 → 個人事業主 → 法人化 のステップで、リスクを最小化しながら進める

---

## 全体ロードマップ

```
Phase 0: 知識をつける（今ここ）
    ↓
Phase 1: PMF検証（プロダクトの価値を確認）
    ↓
Phase 2: 個人事業主として開業（お金を稼ぐ準備）
    ↓
Phase 3: 事業成長（ユーザー獲得・収益化）
    ↓
Phase 4: 法人化（スケールする準備）
```

## Phase 0: 起業の基礎知識をつける

**ゴール**: 起業に必要な知識の全体像を理解し、不安を減らす

### 0-1. 法務・手続きを理解する

| やること | 内容 | 費用 |
|---------|------|------|
| 個人事業主の手続き | 開業届＋青色申告承認申請書を税務署に出すだけ | 無料 |
| 法人化の違い | 合同会社（約8〜12万円）vs 株式会社（約25〜30万円） | — |
| 法人化のタイミング | 利益が年800〜1,000万円を超えたら検討 | — |

**ポイント**: いきなり法人にしなくていい。個人事業主で十分始められる。

### 0-2. お金のことを理解する

| やること | 内容 |
|---------|------|
| 初期費用を把握 | 個人事業主ならほぼゼロ（Apple Developer 12,800円/年 + Google Play $25） |
| 生活費の確保 | 最低6ヶ月分（理想12ヶ月分）= 120〜360万円 |
| 会計ソフト選び | やよい（初年度無料）→ 慣れたらfreeeに移行が合理的 |
| 青色申告を理解 | 65万円控除のためにe-Tax＋複式簿記が条件 |
| インボイス制度 | BtoC中心なら当面は免税事業者のままでOK |

### 0-3. ヘルスケアアプリの法規制を理解する

| 確認事項 | RehabHEPの場合 |
|---------|---------------|
| 薬機法（医療機器該当性） | 運動指導カード作成 = **非該当の可能性が高い**（診断・治療目的ではないため） |
| 個人情報保護法 | 健康情報は「要配慮個人情報」→ 明示的同意が必要 |
| プライバシーポリシー | ストア公開に必須（作成済み: `docs/privacy-policy.md`） |
| 利用規約 | 「医療行為ではない」旨の免責が重要（作成済み: `docs/terms.md`） |

**やるべきこと**: グレーゾーン解消制度（経産省）で薬機法の該当性を事前確認すると安心。

### 0-4. 使える補助金・助成金を知る

| 補助金 | 上限額 | 特徴 |
|--------|--------|------|
| 小規模事業者持続化補助金 | 50〜200万円 | 販路開拓に使える。個人事業主OK |
| ものづくり補助金 | 750〜1,250万円 | 革新的サービス開発向け。審査厳しめ |
| 自治体の創業助成金 | 地域による（東京は上限300万円） | 助成率2/3が多い |
| 日本政策金融公庫 | 最大3,000万円 | 無担保・無保証人の創業融資。最も現実的 |

**注意**: 補助金は後払い（立替必要）。採択率30〜50%。

## Phase 1: PMF検証（まずここから！）

**ゴール**: 「このアプリにお金を払う人がいるか」を確認する

### やること

1. **身近なPT 10〜20名に無料で使ってもらう**
   - 同僚、PT系SNSコミュニティで呼びかけ
   - 2〜4週間使ってもらう

2. **Sean Ellisテストで測定する**
   - 質問: 「このアプリが使えなくなったらどう感じますか？」
   - 40%以上が「非常に困る」と答えたら → PMF達成

3. **フィードバックを集める**
   - 何が便利か、何が足りないか
   - お金を払ってもいいか、いくらなら払えるか

### RehabHEPの強み（差別化ポイント）

競合（リハサク、Rehab Cloud、リモリハ）がデジタル体験に寄っている中で：
- **紙カード特化** = デジタル不慣れな高齢患者にそのまま渡せる
- **シンプルさ** = 複雑な設定不要、すぐカード作成
- **ローカルオンリー** = 個人情報の外部送信リスクなし

## Phase 2: 個人事業主として開業

**ゴール**: 正式に事業として収益を得る体制を作る

### 手続きチェックリスト

- [ ] 開業届を税務署に提出（事業開始から1ヶ月以内）
- [ ] 青色申告承認申請書を提出（開業から2ヶ月以内）
- [ ] 屋号付き銀行口座を開設
- [ ] 会計ソフトを導入（やよい or freee）
- [ ] 事業用クレジットカード作成（経費管理を楽にする）

### マネタイズ設計（B2Bモデル）

```
個人PT: 無料 + 広告あり
  → ユーザー拡大 + 広告でランニングコスト回収
  → 患者が見るのは印刷された紙だけなので広告OK
    ↓
施設プラン: 月2,000〜3,000円（広告なし）
  → 複数スタッフ利用、テンプレート共有、施設ロゴ入り出力、
    データバックアップ・引き継ぎ、利用統計
  → 広告非表示も付加価値の一つ
  → 競合RiGAKUサポ（月約2,000円）と同価格帯
```

戦略: 個人PTが無料+広告で使う → 「施設全体で使いたい」 → 施設契約（Slack/Notion型の拡散モデル）

## Phase 3: 事業成長

**ゴール**: ユーザー数と収益を伸ばす

### マーケティング戦略

| チャネル | アクション |
|---------|-----------|
| SNS（X） | PT向け事例共有、使い方Tips投稿 |
| SEO | 「自主トレ メニュー 作り方」等の記事を書く |
| 学会・研修会 | ブース出展、LT発表（リハ系学会） |
| 施設営業 | リハ科主任・管理者への直接提案 |

### Web版展開
- 施設のPC環境で使えるようにする（印刷はWebの方が相性が良い）
- Expo Web or 別途Webアプリとして構築

### 収益目標の目安
- まず月10万円（個人Pro × 100人 or 施設プラン × 10施設）
- 次に月30〜50万円 → チーム化検討ライン

## Phase 4: 法人化

**タイミングの目安**:
- 利益が年800〜1,000万円を超えた
- B2B取引で法人格が求められるようになった
- 人を雇いたくなった

### 合同会社 vs 株式会社

| 項目 | 合同会社 | 株式会社 |
|------|---------|---------|
| 設立費用 | 約8〜12万円 | 約25〜30万円 |
| 信用度 | やや低い | 高い |
| 向いている場合 | スモールチーム | 資金調達・上場を見据える |

**推奨**: まず合同会社。VCから調達する段階で株式会社に組織変更。

### 最初の採用
- **カスタマーサクセス担当**（PT資格保持者を業務委託）
- ユーザー対応と施設営業を任せる
- 開発は引き続き自分 + AIツール
